What is the Document Object Model (DOM)?

Ans: The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects. Each part of the document—such as elements, attributes, and text—is represented as a node in this tree, allowing programming languages like JavaScript to interact with and modify the document dynamically.

How do select an element from the DOM?

Ans: To select an element from the DOM, can use JavaScript methods that allow to target specific elements based on their tag, class, ID, or other attributes.

  • By ID: document.getElementById()
  • By Class: document.getElementsByClassName()
  • By Tag: document.getElementsByTagName()
  • CSS Selector (Single): document.querySelector()
  • CSS Selector (Multiple): document.querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

Ans: Event delegation is a technique in the DOM where a single event listener is added to a parent element to handle events for its child elements. Instead of attaching individual event listeners to multiple child elements, the event is captured at the parent level and then delegated to the target child element through event propagation (bubbling).

How do manipulate an element's attributes and styles using the DOM?

    Ans:
  • Attributes: Use getAttribute(), setAttribute(), removeAttribute() to work with attributes.
  • Inline Styles: Use the style property to modify inline styles directly.
  • CSS Classes: Use classList to add, remove, or toggle classes efficiently.
